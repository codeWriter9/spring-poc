# Spring-Boot-HelloWorld

Starter Projects to  kick start a spring boot with log4j config

## Getting Started

Do a git clone 


### Prerequisites

You should only need a git client like git bash, an IDE, maven 3.x and certainly the Java platform with sdk


## Running the tests

and for hosting
